Quickadoo (Midterm Project)

MVP Web Application - Lighthouse Labs

A 5-day Minimum Viable Product for Lighthouse bootcamp midterm. Quickadoo is a web app intended to help users quickly and easily schedule events within a group. It allows users to cast votes on their preferred options to determine the best time and date to meet!

Tech Stack: Javascript/jQuery, NodeJS, Express, Knex, Sass, EJS
