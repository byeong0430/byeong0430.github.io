Quickadoo (Midterm Project)

MVP Web Application - Lighthouse Labs

GitHub Site

- A 5-day Minimum Viable Product for Lighthouse bootcamp midterm. Quickadoo is a web app intended to help users easily schedule events. It allows users to cast votes on their preferred options to determine the best time and date to meet!

- I participated in all aspects of the web app: front-end and database designing & back-end route set-up.

Tech Stack: Javascript/jQuery, NodeJS, Express, Knex, Sass, EJS
